What Changed
v1.0.8 Changelog

Stability and refined combat logic.

01

Velocity Hook Fixed

Resolved a major bug where the Velocity module was not correctly hooked to network packets. Knockback reduction now functions as intended.

02

Combat Module Overhaul

Expanded AimAssist and AutoClicker weapon detection to include Axes, Maces, and Tridents. Refined target filtering.

03

Logger System

Implemented a new centralized Logger to replace silent failures and System.err calls, improving overall release stability.

04

Rendering & Null-Safety

Added extensive null-safety checks in core modules and polished the billboarded quad health bar system.
