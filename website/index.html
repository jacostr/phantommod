What Changed

v1.0.5 sharpens the things you touch the most.

Profiles Feel Safer

Saving over an existing slot now asks for confirmation instead of replacing it immediately.

HUD Flow Is Cleaner

HUD settings are now in their own bottom-right button instead of cluttering the normal module list.

Module Browsing Is Cleaner

The ClickGUI now feels more streamlined with the HUD controls pulled out into their own corner button.

Scaffold Was Simplified

The scaffold module was stripped back to regular synced placement without extra settings noise.

Install Route

Five quick steps and you are in-game.

  1. Install Fabric Loader for Minecraft 1.21.11.
  2. Install the matching Fabric API build.
  3. Download the PhantomMod release jar.
  4. Move it into .minecraft/mods.
  5. Launch Minecraft using the Fabric profile.
